# Ted Voca — 핵심 화면 와이어프레임

> ASCII 와이어프레임 5종. P0~P5 구현 시 UI 레퍼런스.

---

## 1. 온보딩 (Onboarding)

3-step wizard. Ted 마스코트가 각 단계에서 안내.

```
┌─────────────────────────────┐
│  ● ○ ○                      │  ← progress dots
│                             │
│      [ Ted mascot ]         │
│                             │
│  "안녕! 나는 Ted야.         │
│   영어 목표가 뭐야?"        │
│                             │
│  ┌─────────────────────┐    │
│  │ 📚 시험 (토익/수능)  │    │  ← goal cards (single select)
│  ├─────────────────────┤    │
│  │ 💬 일상 회화         │    │
│  ├─────────────────────┤    │
│  │ 💼 비즈니스          │    │
│  └─────────────────────┘    │
│                             │
│  ┌─────────────────────┐    │
│  │      다음 →          │    │
│  └─────────────────────┘    │
└─────────────────────────────┘

Step 2: 일일 목표
┌─────────────────────────────┐
│  ○ ● ○                      │
│  "하루에 몇 분 공부할래?"   │
│                             │
│   [ 5분 ]  [ 10분 ]  [ 20분 ]│  ← pill buttons
│                             │
│  ┌─────────────────────┐    │
│  │      다음 →          │    │
│  └─────────────────────┘    │
└─────────────────────────────┘

Step 3: 확인
┌─────────────────────────────┐
│  ○ ○ ●                      │
│  "좋아! 준비됐어 🎉"        │
│                             │
│  목표: 시험 (토익)          │
│  일일: 10분                 │
│                             │
│  ┌─────────────────────┐    │
│  │   학습 시작하기       │    │
│  └─────────────────────┘    │
└─────────────────────────────┘
```
